Полезные сочетания клавиш для Visual Studio

Ctrl + F5 - запустить программу без отладки
Ctrl + K + C - закомментировать выделенный блок кода
Ctrl + K + U - раскомментировать выделенный блок кода
Ctrl + K + D - форматировать документ
Ctrl + R + R - переименовать переменную во всём документе

Средства -> Настройки -> Команды -> Дополнительные параметры сохранения - кодировка файла кода
